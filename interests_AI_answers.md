### 输入

### 矩阵乘法
计算 `A @ A`（或 `A @ A.T`？代码中是 `np.matmul(x, x.T)`，因此对于方阵，`A @ A.T` 与 `A @ A` 在对称时可能不同，但这里由于是对称的？输入矩阵 `A` 并不对称，但 `A @ A.T` 的结果与 `A @ A` 不同。我们应确认代码中的操作：在 `operations` 中，`matmul` 的 lambda 是 `lambda x: np.matmul(x, x.T) if x.ndim >= 1 and x.size > 0 else None`，所以是矩阵乘以其转置。对于 2×2 矩阵，`A.T` 是：
```
[[ True  True]
 [ True False]]
```

计算 `A @ A`：
- (0,0) = A[0]·A[0] = True*True + True*True = 1+1 = 2 → 在 NumPy 中非零为 True，在 PyTorch 中保持整数 2。
- (0,1) = A[0]·A[1] = True*True + True*False = 1+0 = 1 → NumPy: True，PyTorch: 1。
- (1,0) = A[1]·A[0] = True*True + False*True = 1+0 = 1 → 同上。
- (1,1) = A[1]·A[1] = True*True + False*False = 1+0 = 1 → 同上。

```
 [1 1]]
```

```
[[ True  True]
 [ True  True]]
```

